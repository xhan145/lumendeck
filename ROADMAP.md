# LumenDeck Roadmap

A living, non-binding sketch of where LumenDeck is heading. Items are directional, not
promises; priorities shift with community feedback and contributions. See
[CONTRIBUTING.md](CONTRIBUTING.md) if you would like to help with any of these.

## Near-term polish

- Dashboard and page-level UI refinement across Controls, Settings, and Gallery.
- Native folder pickers for the model/LoRA/output/cache directory fields.
- In-app runtime repair for the managed Diffusers runtime (today it is a manual delete).

## Constellation system improvements

- Smoother orb/ghost motion and richer node constellations in the 3D graph.
- Better anchor management and path-recording ergonomics.
- Performance work so large graphs stay fluid on mid-range GPUs.

## Gradient engine and timeline

- Gradient engine polish: more controllable falloff, blending, and re-tint behavior.
- Finer timeline positioning for motion keyframes and clip editing.
- Audio-reactivity mapping presets.

## Preset packs

- Curated prompt/sampler preset packs and `.lumen` starter workflows.
- Community-submitted preset gallery with sample renders.

## Docs and tutorials

- Task-oriented tutorials: first real render, motion clips, evolve runs, audio reactivity.
- Deeper troubleshooting guides building on `docs/diagnostics.md`.

## Installer and release polish

- Streamlined Windows installer and updater experience.
- Smaller download and faster first-run for the bundled bridge.

## Community contributions

- Issue templates and a good-first-issue backlog.
- Making preset and docs contributions as low-friction as possible.
